Database Development - Assignment 2

Assignment Overview

In this assignment, I used Microsoft Access to create the relational database that I had planned in Assignment 1. I built the database structure, entered data, created forms and reports, and implemented queries to retrieve useful information. I also tested the database to ensure it worked correctly and met the user requirements.

What I Did

I started by creating a new database in Microsoft Access. I set up the required tables, carefully choosing appropriate data types and setting primary keys. I also defined relationships between the tables using one-to-many links where needed, ensuring referential integrity was enforced.

I then entered sample data into each table to allow me to test the database and demonstrate its functionality.

I created a range of queries to search, sort, and filter data. I also built custom forms to make data entry easier for users and designed professional-looking reports to present key information clearly.

Testing

I tested my database to make sure it worked as required. I checked that:

  • Tables stored data correctly and enforced data validation rules
  • Relationships maintained data integrity
  • Queries returned the correct results for a range of test scenarios
  • Forms allowed data to be entered and updated easily
  • Reports displayed accurate and useful information

I also tested for error handling and unusual input, such as blank fields and invalid data.

Evaluation

I evaluated my database against the original user requirements and concluded that it met all key goals. The database was easy to use, reliable, and produced correct outputs. If I were to improve it further, I could add more advanced features such as calculated fields, additional validation rules, and more complex queries and reports to meet advanced user needs.
