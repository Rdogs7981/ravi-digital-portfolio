Database Development - Assignment 1

Assignment Overview

In this assignment, I focused on designing and planning a relational database to meet a specific set of user requirements. I analysed the brief, identified what data needed to be stored, and designed a suitable database structure. I also planned queries, forms, and reports that would be required in the finished database.

What I Did

I began by reading the project brief and identifying the types of data that would need to be stored. I planned how the data would be organised into tables and identified relationships between those tables (one-to-many, many-to-many, etc.). I used an Entity Relationship Diagram (ERD) to visualise the database structure.

I also planned which queries, forms, and reports would be needed by the users to interact with the database and view key information. I documented how the database would be designed to ensure data accuracy, consistency, and security.

Skills Gained

I developed skills in:

  • Analysing user requirements for a database
  • Planning tables, fields, data types, and relationships
  • Creating Entity Relationship Diagrams (ERDs)
  • Designing forms and reports for data entry and output
  • Planning validation rules and data integrity techniques
  • Documenting database design clearly and professionally
