Software Development - Assignment 1

Assignment Overview

In this assignment, I focused on the design and planning stage of a software project. The goal was to plan a piece of software based on a set brief and user requirements. I produced design documentation and visual plans for how the software would function and look before moving on to the development stage.

What I Did

I started by analysing the brief to understand what the software needed to do. I then identified the target users and the purpose of the software. After that, I created a detailed plan, including flowcharts and screen designs/mockups. I also considered user interface design, usability, and accessibility when planning how the software would work.

Skills Gained

I developed skills in:

  • Understanding and analysing user requirements
  • Planning the structure of software using flowcharts
  • Designing clear and user-friendly interfaces
  • Documenting design choices and explaining reasoning
  • Planning for testing and evaluation of the software
