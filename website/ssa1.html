Spreadsheet Development - Assignment 1

Assignment Overview

In this assignment, I focused on the design and planning stage of a spreadsheet project. The goal was to plan a spreadsheet that meets a specific set of user requirements. I documented the purpose, structure, and key features of the spreadsheet before moving on to the actual creation stage.

What I Did

I began by reading the brief and identifying what the spreadsheet needed to do. I defined the target users and created a design specification. I planned the structure of the spreadsheet, including what worksheets would be needed, what data would be stored, and what formulas and functions would be used.

I also created mockups and layout diagrams to show how the spreadsheet would look. I considered how to make the spreadsheet easy to use and how to format it so that key information would stand out clearly.

Skills Gained

I developed skills in:

  • Analysing user requirements for a spreadsheet
  • Planning spreadsheet structure and layout
  • Choosing appropriate formulas and functions
  • Designing for clarity and ease of use
  • Documenting design decisions and presenting design plans
