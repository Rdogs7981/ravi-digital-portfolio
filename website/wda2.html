Website Development - Assignment 2

Assignment Overview

In this assignment, I built the website that I had previously designed in Assignment 1. Using HTML and CSS, I created the structure, content, and style of the website. The goal was to produce a working, fully functional multi-page website that meets the needs of the target audience.

What I Did

I used the wireframes and design ideas from Assignment 1 as a guide. I wrote the website code using HTML to structure the content and CSS to style the pages. I created a clear navigation system and ensured all links worked correctly. I added images, text, and other content while making sure that the design matched my original plan.

I tested the website on different browsers and devices to check for compatibility and performance issues. I also made improvements based on feedback.

Skills Gained

I developed skills in:

  • Writing valid HTML and CSS
  • Creating consistent layouts across multiple pages
  • Using images, fonts, and colours effectively
  • Building navigation menus
  • Making sure the website works on different devices (responsive design)
  • Testing and debugging web pages
