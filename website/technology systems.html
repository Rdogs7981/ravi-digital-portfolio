Technology Systems

This unit helps you understand how technology systems work and how the different components—like hardware, software, and networks—connect and function together. It builds knowledge of how data is stored and processed and how different systems can be used in personal, educational, and business environments.

This unit is assessed through an external online exam.

What You'll Learn:

1. Hardware Components

  • Processor (CPU) – carries out instructions.
  • RAM – provides temporary working memory.
  • Storage – including HDDs, SSDs, and optical drives.
  • Input Devices – mouse, keyboard, scanner.
  • Output Devices – monitors, printers, speakers.

2. Software

  • Operating systems (Windows, macOS, Linux).
  • Utility software (like antivirus, backup tools).
  • Application software (such as Word, Excel, web browsers).
  • Drivers – help hardware and software communicate.

3. Networks and Connectivity

  • Types of networks: LANs and WANs.
  • Network hardware: routers, switches, modems.
  • Connections: Wi-Fi, Ethernet, Bluetooth.
  • Resource sharing: how networks allow data sharing.

4. Data Storage

  • Binary data – how computers use 1s and 0s.
  • File types: .txt, .mp3, .jpg, .pdf.
  • Compression: why and how files are reduced in size.

5. System Performance and Troubleshooting

  • Performance factors: CPU speed (GHz), RAM size, storage capacity, background processes.
  • Troubleshooting: dealing with slow performance, software errors, etc.

Assessment

You’ll take an externally set, computer-based exam. The test includes multiple-choice, short-answer, and sometimes scenario-based questions. You’ll need to demonstrate an understanding of both the technical side of systems and how they’re used in real-world situations.
